# MindOrigin / P/G Theory — Document Guide

This folder collects the MindOrigin (P/G Theory) materials: background, the three
core papers, and the economics extensions.

Notation used throughout: **P** = an entity's property state (safety, resources,
capability…); **G** = the relationship field between entities (trust, obligation,
threat…); **ΔP / ΔG** = the change an action produces in each.

## 1. Background and research notes
Read these first — how the project began and its core idea.

- **MindOrigin_PG_research_review.pdf** (English) — full background and the central
  thinking behind the project. This is the broader English version and also contains
  the project analysis.
- **MindOrigin_Project_Analysis.pdf** (Chinese) — the project analysis in Chinese;
  the same analysis appears in English inside the research review above.

*Usage: feed both to an AI together to provide the complete background.*

## 2. Core papers
The three papers that define the framework.

- **mindorigin_pg_theory.pdf** — *Technical foundation.* Defines actions as
  transformations that change property states (P) and relationship fields (G), and
  specifies a concrete training pipeline for the estimator that predicts those
  ΔP/ΔG changes.
- **PG_Valuation_Relational_Representation_Layer.pdf** — *Behavioral breadth.* Argues
  the same P/G representation explains why different agents act differently under
  identical conditions (the Behavioral Diversity Principle), while insisting that
  describing behavior is not the same as authorizing it.
- **pg_minimal_affective_core.pdf** — *Emotion.* Claims a primitive good/bad feeling
  can be generated from effective P change, with G deciding whose P the agent
  actually feels.

## 3. Economics extensions
How the framework could be applied to, and seed new work in, economics. Read after
the core papers.

- **PG_Innovation_Frontiers_ECON.pdf** — *High-level directions.* The latest overview
  of where P/G could open new ground across economic fields, with the key obstacle
  for each.
- **PG_Theory_Economics_Research_Agenda.pdf** — *Detailed companion.* Turns those
  directions into specific candidate papers — each with its existing literature and
  references, a realistic publication venue, and the identification hurdle it must
  clear.

## Suggested reading order
Section 1 (background) → Section 2 (core papers) → Section 3 (economics).
